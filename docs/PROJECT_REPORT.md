# Отчет по проекту Site Junona

(Перемещено в docs для структурирования документации.)

## Обзор проекта

**Site Junona** - это веб-приложение на Flask для управления информационными страницами образовательного учреждения. Проект включает систему управления файлами, мастер заполнения и автоматическую очистку несуществующих файлов.
